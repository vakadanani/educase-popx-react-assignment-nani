# Educase PopX React JS Task

React + JavaScript implementation of the PopX mobile interface from the Adobe XD design.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

- Landing screen
- Login screen
- Signup screen
- Account settings/profile screen

## Navigation

- Create Account -> Signup
- Already Registered? Login -> Login
- Login -> Profile
- Signup Create Account -> Profile
